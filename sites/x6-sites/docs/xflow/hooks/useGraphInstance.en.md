Hook for obtaining the canvas instance

## Basic Usage

```tsx
 const graph = useGraphInstance();
```

## API

```tsx
  const graph: Graph = useGraphInstance()
```

## Return Value

| Parameter | Description   | Type   | Default Value |
|-----------|---------------|--------|---------------|
| graph     | Canvas instance | `Graph` | `null`        |

## Parameters

None

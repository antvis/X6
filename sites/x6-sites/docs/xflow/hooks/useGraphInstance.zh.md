用于获取画布实例的 Hook

## 基础用法

```tsx
 const graph = useGraphInstance();
```

## API

```tsx
  
  const graph: Graph = useGraphInstance()

```

## 返回值

| 参数 | 描述 | 类型 | 默认值 |
|--------|------|------|-------|
| graph | 画布实例 | `Graph` | `null` |

## 参数

无

# 安装

:::info{title="前提条件"}

- 熟悉命令行
- 已安装 18.0 或更高版本的 Node.js

:::

## 通过包管理器

```bash [NPM]
# 使用 npm
$ npm install @antv/xflow --save

#  使用yarn
$ yarn add @antv/xflow

# 使用pnpm
$ pnpm add @antv/xflow
```

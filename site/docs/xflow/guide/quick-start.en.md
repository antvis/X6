# Installation

:::info{title="Prerequisites"}

- Familiarity with the command line
- Node.js version 18.0 or higher installed

:::

## Using a Package Manager

```bash [NPM]
# Using npm
$ npm install @antv/xflow --save

# Using yarn
$ yarn add @antv/xflow

# Using pnpm
$ pnpm add @antv/xflow
```
